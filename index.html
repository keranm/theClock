The Clock Project

Setting out to build a number of different clock faces using where possible nothing but CSS. These projects will use jQuery to update css to create the illusion of animation, however all placement, layout and UI will be created using CSS

Analog Clock

The first cab of the ranks is a simple analog clock, with a rolex style smooth second hand animation.

Airport Style

The next clock I want to make is one that uses a flipboard style face. You know those airport type signs with the split and pane that falls/rotates to display content? Yeah one of those
